# v0.5.1 - NPM Distribution Readiness

This patch release prepares deterministic-agent-system for npm distribution.

## Highlights

- Enables npm publication by setting private to false.
- Keeps the package name as deterministic-agent-system.
- Preserves the det-agent bin entry.
- Includes selected operational PowerShell scripts in the npm package.
- Keeps local runtime data, backups, and local environment files excluded.
- Validates npm packaging through npm pack and npm publish dry-run.

## Package behavior already validated

- Installation from generated .tgz.
- npx det-agent help from installed package.
- npx det-agent backup -KeepLast 1 from installed package.
